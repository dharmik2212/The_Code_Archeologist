# Code Archeologist (VS Code Extension)

Ask questions about your workspace and get:
- Relevant code snippets (retrieved via embeddings)
- Git blame attribution (who wrote the relevant lines), including unsaved editor changes
- A final answer generated by a Hugging Face model

## Setup

- `npm install`
- `npm run compile`

## Run / Debug

Press `F5` to launch an Extension Development Host.

## Usage

- In VS Code Chat, pick the participant **Code Archeologist** and ask a question.
- Commands:
  - **Code Archeologist: Set Hugging Face Token**
  - **Code Archeologist: Reindex Workspace**

## Defaults

- Embeddings model: `sentence-transformers/all-MiniLM-L6-v2`
- Answer model: `meta-llama/Llama-3.1-8B-Instruct`
